centOs命令总结

发表于 2019-05-10 更新于 2021-03-26 分类于 -Linux

Linux代码指令

pwd (print work directory) 显示目前所在目录

mkdir (make directory) 创建一个新的目录

mkdir -p (帮助你直接将所需要的目录全部递归创建)

rmdir (remove directory) 删除一个空目录

cp (cp file) 复制文件或目录

rm (remove) 移除文件或目录

mv (move file) 移动文件与目录

命令代码 意思
ls 查询当前目录下的所有文件
cd /*/ * 跳转文件夹
mkdir /usr/local/java/ 在/usr/local/下面创建java文件夹
mv 移动指令
rm -f 删除某个文件
rm -rf 逐次删除文件夹
tar -zxvf jdk-8u171-linux-x64.tar.gz -C /usr/local/java/ 解压到新建目录
vim /etc/profile 打开文件
javac hello.java
java hello		 运行java程序hello.java
cd/etc —– vim profile —-按下i才能进入编辑状态（有时候需要加enter？不知道为什么？）—- 退出命令（:wq） —– :q!是不保存文件退出
cat /proc/version 查看内核版本
按ESC键 跳到命令模式，然后：
:w 保存文件但不退出vi
:w file 将修改另外保存到file中，不退出vi
:w! 强制保存，不推出vi
:wq 保存文件并退出vi
:wq! 强制保存文件，并退出vi
q: 不保存文件，退出vi
:q! 不保存文件，强制退出vi
:e! 放弃所有修改，从上次保存文件开始再编辑
pwd 显示目前的

cent OS配置Java环境

  1. 下载 jdk（一定要使用国内镜像 – 贴一个华为云的镜像网址 – https://repo.huaweicloud.com/java/jdk/）

  2. 创建/usr/local/java/ (注意：操作文件时最后不能有/，这个/是针对文件夹的)

  3. 将下载的文件用Xftp放到/usr/local/java/文件夹下面

  4. 解压 tar -zxvf ***（注意可以用tab来自动补全文件名）

  5. 编辑 JAVA_HOME (– vim /etc/profile – 输入以下环境配置语句)

  6. 编译一下 （source /etc/profile）

  7. 测试一下 （java -version javac -version）

cent OS配置tomcat服务器

  1. 依然是下载文件
  2. 和上面步骤一样 导入-解压
  3. 修改一下tomcat文件夹名字，为了以后输入方便 mv tomcat8.5.50/ tomcat
  4. 测试一下安装是否成功 首先进入bin文件夹 – cd /usr/local/tomcat/bin – 然后运行 – sh startup.sh
