JSP技术

发表于 2019-05-09 更新于 2021-01-24 分类于 -Java

JSP技术：（ Java Server Pages）

​ 特点：

​ 1、本质上还是Servlet(Java类)

​ 2、跨平台，一次编写运行

​ 3、组件跨平台

​ 4、 健壮性和安全性

Jsp的访问原理：

​ tomcat服务器实际上依然在执行Servlet，我们编写的 *.jsp 文件实际上被转译为同名的 *.jsp.java类（不同软件的执行方式可能不同，但是实质上都是转译为相应的java类（extends HttpServlet）去执行）

Jsp的page指令：

​ <%@page 属性名=”属性值” 属性名=”属性值”…%>

​ language：声明jsp要被转译的语言

​ import：声名转译的java文件要导入的包，不同的保用逗号(,)隔开

​ pageEncoding：声明编码格式

​ contentType=”text/html”; charset=”utf-8” 设置jsp数据响应给浏览器时，浏览器的编译格式

​ session：设置转译的servlet中是否开启session支持，默认(true)开启

​ errorPage：设置jsp运行错误跳转的界面

​ extends：设置jsp编译后的文件要继承的父类
