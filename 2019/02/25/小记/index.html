小记-游人

发表于 2019-02-25 更新于 2021-05-03 分类于 -Life

序

大半夜睡不着爬起来打着闪光灯找纸笔，然后在初春的寒意里迅速缩回被窝真是一件让人开心的事情！（小记一下自贡游）

话说不明不白就上了豪狗贼船的我早记不得是怎样答应了这弱智的元宵灯会游。真是一件神器的事情哈，上午我还我在实验室迷迷糊糊的码代码，下午一个电话过来我就突然转场奔赴荣县了？？？

一向对于“游人”这个词汇不太感冒的我，站在灯会面前也会不禁感叹！一年一度的元宵灯会——传承千年的习俗，在自贡这个神奇的地方再一次惊艳了我们。想想古人游灯赏花又是怎样一番景色呢？会有如今的灯火璀璨吗？那时的文人雅客，稚子幼童，又会有怎么样的别情雅致呢？

（一）起

好像出游也是一件让人开心的事情，快乐的人们做着漫无边际的事的快乐人生。孤僻一词虽然渐渐离我更近，但身边有这样一群弱智少年郎陪伴还是一件让人会想要咧嘴笑的事情！

那迷迷糊糊的三人组就开始这迷迷糊糊的旅途吧！

——“古人云，三人行，必有我师焉，择……”

——打断，三人行，必有狗焉，择豪狗行之，顺李铭爷爷心意！

（二）夜宿

夜宿荣县。

我：老板，两个人，标间多少钱啊？

老板：现在标间148块（迟疑ing）。

（老板持续迟疑中)

老板：这样，我给你们升级成亲子房行吗？

我+陈豪：亲子房（一脸疑惑ing）？？？

老板：是这样的，亲子房会更舒服点，只是一个床大点，一个床小点（语气逐渐虚弱 ）……

我：成了，我睡大的，我儿子睡小的。

老板：（憋住笑ing)。

陈豪：儿子？？？我是你爹！我拒绝这个提议！

我：老板我给钱，您听我的，就亲子房（十分开心ing）！！！

二人行，必有我子，择亲子套而入，善莫大焉（这该死的文言文）！

（三）荣县灯会VS自贡灯会

恕我直言，没有大佛的荣县灯会，在自贡等会面前就是个弟弟！

​ ——花狗

自贡的灯会有点好看！我有时想古人游园赏灯猜谜过元宵应该是一件很有意思的事情——春寒未歇，携童子二三，呼朋唤友，或同女眷，赏灯猜谜。

欧阳修有首写元宵的绝唱：去年元宵夜，花市灯如昼，月上柳梢头，人约黄昏后。今年元夜时，月与灯依旧。不见去年人，泪湿春衫袖。可如今这灯光璀璨的灯会，又会让文人雅客说道些什么呢？本是天上星，如坠人间二月天？二月天气冷冷清清，花市灯如昼倒是也切切实实，人影憧憧也是人影憧憧，来年人依然是来年人，以后会是去年人吗？

（四）自贡人均变态辣？？？

豪狗无情无义抛弃我、李掐睿二人，同他那个不知某年某月某日认识的某姓学长胡吃海喝。

我和李掐睿只好去两人觅食，然后我们在冷吃兔和火锅之间选择了冷吃兔。

“这兔子还挺卖相，嚯嚯嚯，爷开动了！”

……

……

无止境的沉默 + 可乐 + 冷茶 + 加多靌 + 洗脸漱口 + 抖腿 + 像只傻狗坐在座位上捏紧双拳小幅度迅速摆动 + 拳头疯狂捶腿 + 辣到屏息！

我去你他谁妈的自贡冷吃兔++++挺好吃（此条+999）+ 我该死的菊花？？？
